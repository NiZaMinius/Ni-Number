## Концепция константы квантового рассеяния энергии (Число Ни)

**Автор гипотезы:** NiZaMinius

### 1. Определение и фундаментальный смысл

Константа квантового рассеяния $\eta_{\nu}$ **(Число Ни)** — это фундаментальная математическая величина, определяющая суммарную амплитуду затухания калибровочных полей на субпланковских расстояниях.

Значение Числа Ни строго фиксировано математически и приблизительно равно: **1,8893766...**

В современной физике предполагается существование скрытых, компактифицированных пространственных измерений (многообразий Калаби-Яу). Данная концепция утверждает, что при сближении частиц на дистанции меньше размеров атомного ядра часть виртуальных бозонов (носителей взаимодействия) рассеивается в этот скрытый объём. Число Ни выступает точной мерой доли энергии, которая покидает трёхмерную брану.

---

### 2. Математическая структура константы

Константа задаётся нормированным рядом по всем спектральным модам Калуцы-Клейна:

$$\eta_{\nu} = \sum_{n=1}^{\infty} \frac{\pi^n}{n! \cdot 2^{n^2}}$$

Индекс $n$ (1, 2, 3...) в данном ряду обозначает не порядковый номер скрытого измерения (так как все они существуют одновременно), а номер **квантовой моды** (энергетического уровня). Подобно тому как гитарная струна выдаёт набор гармоник, кванты полей резонируют внутри единого многомерного многообразия на дискретных частотах. Бесконечный ряд суммирует все возможные квантовые состояния утечки энергии.

Каждый элемент формулы имеет строгое обоснование в дифференциальной геометрии и теории струн:

#### 2.1. Геометрия объёмных резонансов (числитель $\pi^n$)

Степени числа $\pi$ определяют геометрический фактор многомерности. Рост степени показывает, как расширяется площадь фронта волны при проникновении энергии вглубь новых координат на каждой последующей гармонике.

#### 2.2. Топологическая симметрия (факториал $n!$)

На первый взгляд, появление факториала может показаться математической подгонкой. Однако это прямое следствие дифференциальной геометрии комплексных кэлеровых многообразий (пространств Калаби-Яу).

Чтобы вычислить физический объём такого многообразия, необходимо проинтегрировать его фундаментальную форму объёма, которая строится из кэлеровой 2-формы ($\omega$). Точная формула объёма выглядит так:

$$V = \int_{M} \frac{\omega^n}{n!}$$

Из-за антикоммутативности внешнего произведения (оно учитывает направления осей) при умножении возникает огромное количество перекрывающихся перестановок координат. Математика требует разделить интеграл на количество этих перестановок ($n!$). Таким образом, факториал в Числе Ни естественным образом **«схлопывает» избыточные симметрии** свёрнутого пространства, не давая объёму и утекающей энергии разрастаться бесконечно.

#### 2.3. Квантовая блокировка (основание $2$ и степень $n^2$)

В физике непрерывных процессов за затухание обычно отвечает число Непера ($e$). Однако на субпланковских масштабах пространство дискретно.

В матричных моделях М-теории (например, BFSS) взаимодействия между бранами описываются матрицами размера $n \times n$, где $n$ — квантовое число моды. Размер матрицы, $n^2$, определяет количество степеней свободы квантового канала. Поскольку на фундаментальном уровне квантовая информация бинарна (кубит: наличие связи или её отсутствие, спин вверх/вниз), двойка выступает базисом состояния.

Полное число микросостояний равно $2^{n^2}$. Использование экспоненты $e$ означало бы применение классической статистики к квантовой гравитации. Двойка — это подпись фундаментальной информационной природы пространства. Квадратичный рост связей вызывает лавинообразное сопротивление на высоких частотах, обеспечивая **сверхбыструю сходимость** ряда.

### 3. Модификация законов взаимодействия на сверхмалых расстояниях

Согласно данной гипотезе, классические законы обратных квадратов (Ньютона и Кулона) ломаются на субпланковских дистанциях, как это предсказывается в моделях с дополнительными измерениями (например, теории Рандалла-Сундрума).

Число Ни ($\eta_{\nu}$) не просто делит силу пополам в макромире — оно работает как **динамический интегральный коэффициент** в самом уравнении поля, выступая амплитудой перехода между браной (3D-миром) и балком (высшим измерением).

Модифицированный закон силы выглядит так:

$$F(r) = \frac{G \cdot m_1 m_2}{r^2} \cdot \frac{1}{1 + \eta_{\nu} \cdot \left(\frac{R_{\mathrm{Planck}}}{r}\right)^d}$$

где $R_{\mathrm{Planck}}$ — планковская длина, $r$ — расстояние, $d$ — эффективное число скрытых измерений.

**Физический смысл уравнения:**

| Режим | Физика |
|---|---|
| $r \gg R_{\mathrm{Planck}}$ (макромир) | Дробь → 0, знаменатель → 1, классический закон восстанавливается |
| $r \to R_{\mathrm{Planck}}$ (микромир) | Активируется многомерная физика; η_ν задаёт «пропускную способность» квантового портала; переносчики взаимодействия утекают в геометрические поры многообразия, вызывая резкое падение регистрируемой силы |

## Установка

Добавьте в `Cargo.toml`:

```toml
[dependencies]
ni-number = "0.1"
```

### Системные зависимости

**Ubuntu / Debian**
```bash
sudo apt-get install libgmp-dev libmpfr-dev libmpc-dev
```

**macOS (Homebrew)**
```bash
brew install gmp mpfr libmpc
```

**Windows (Git Bash + MSYS2)**

1. Установить [MSYS2](https://www.msys2.org)
2. В терминале MSYS2 MinGW64:
```bash
pacman -S --needed base-devel mingw-w64-x86_64-toolchain m4 make mingw-w64-x86_64-gmp mingw-w64-x86_64-mpfr
```
3. Добавить в PATH в Git Bash (`~/.bashrc`):
```bash
export PATH="/c/msys64/mingw64/bin:/c/msys64/usr/bin:$PATH"
```
4. Переключить toolchain Rust:
```bash
rustup toolchain install stable-x86_64-pc-windows-gnu
rustup default stable-x86_64-pc-windows-gnu
```

## Использование

```rust
use ni_number::{NI_F64, ni_number_digits, ni_number, bits_for_digits};

fn main() {
    // Быстрая константа f64 (~15 знаков)
    println!("η_ν ≈ {:.15}", NI_F64);

    // 100 знаков после запятой в виде строки
    let s = ni_number_digits(100);
    println!("η_ν = {}", s);

    // rug::Float для вычислений произвольной точности
    let eta = ni_number(bits_for_digits(500));
    println!("{:.500}", eta);
}
```

### Запуск примеров

```bash
cargo run --example basic --release
cargo run --example high_precision --release
```

## API

| Функция | Возвращает | Описание |
|---|---|---|
| `NI_F64` | `f64` | Константа двойной точности |
| `NI_F32` | `f32` | Константа одинарной точности |
| `NI_50_DIGITS` | `&str` | Первые 50 знаков после запятой |
| `ni_number(bits)` | `rug::Float` | Значение произвольной точности |
| `ni_number_digits(n)` | `String` | Строка с `n` знаками после запятой |
| `ni_term(n, bits)` | `rug::Float` | Один член ряда с индексом `n` |
| `bits_for_digits(n)` | `u32` | Точность в битах для `n` десятичных знаков |
